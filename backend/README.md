## Response 状态码

| 代码 | 含义 |
| --- | --- |
| 0 | 成功 |
| -1 | 未知错误 |
| -2 | 数据库查询错误 |
| -3 | 数据库写入错误 |
| -4 | Hash错误 |
| -5 | Cookie提取错误 |
| -6 | 解析错误 |
| -101 | Authorization 解析格式错误 |
| -102 | 未提供 Authorization |
| -103 | Authorization 无效/过期 |
| -104 | 未提供 JWT Claims |
| -201 | 未找到 Refresh Token |
| -202 | Refresh Token 过期 |
| -203 | Refresh Token 错误 |
| -204 | Refresh Token 失效 |
| -301 | 用户不存在 |
| -302 | 密码错误 |
