# DTBox

## 简介
是一款用于美股日内交易的工具软件，提供实时行情、数据分析等功能。

## 功能
- 保存布局: 云端保存布局，支持多设备同步。
- 找票系统: 支持L2数据找票
- 实时行情：支持多种数据源，包括股票、期货、期权等，提供实时行情数据。
- 数据分析：提供多种数据分析工具，包括K线图、MACD、RSI等，帮助用户进行技术分析。
